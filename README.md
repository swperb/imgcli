# imgcli — command-line image conversion & processing in C

A dependency-free CLI to **convert, resize, crop, filter, and composite images**
(PNG / JPEG / BMP / TGA / GIF / PPM) via an ffmpeg-style filtergraph. One small
C binary, no system libraries.

> 🚧 Repository scaffold. The full implementation is delivered in the first
> pull request — see the open PRs.
